# 📊 Статус проекта Transformation Map

## 🎯 **Цель проекта:**
Геймификированная платформа для личностных трансформаций и достижения целей с Telegram авторизацией.

## 🚀 **Текущее состояние:**
- ✅ **Домен:** https://transformation-map.com (работает с SSL)
- ✅ **Сервер:** Ubuntu, Nginx + PM2
- ✅ **Стек:** Next.js 15, TypeScript, Tailwind CSS
- ✅ **Авторизация:** Telegram Web App + NextAuth
- ✅ **Интернационализация:** ru/en языки
- ✅ **Деплой:** Настроен автоматический workflow

## 📁 **Структура проекта:**
```
/var/www/transformation-map/
├── app/                    # Next.js App Router
│   ├── [locale]/          # Интернационализация
│   │   ├── page.tsx       # Главная страница
│   │   ├── auth/          # Страницы авторизации
│   │   └── protected/     # Защищенные страницы
│   ├── api/               # API routes
│   └── globals.css        # Глобальные стили
├── components/            # React компоненты
│   ├── ui/               # UI библиотека
│   ├── theme-switcher.tsx # Переключатель темы
│   └── LanguageSwitcher.tsx # Переключатель языка
├── lib/                  # Утилиты
├── messages/             # Переводы (ru.json, en.json)
└── COMMANDS.md           # Команды для разработки
```

## 🔧 **Настроенные команды:**
```bash
# Разработка на сервере:
npm run dev:start     # Dev режим с hot reload
npm run save          # Быстрое сохранение в Git
npm run prod:start    # Продакшен через PM2

# Управление:
npm run pm2:status    # Статус приложения
npm run pm2:logs      # Логи приложения
npm run deploy        # Полный деплой
```
